# Dockerized Two-Tier Flask App

## About
Two-tier Flask application with MySQL using Docker and Docker Compose.

- Dockerfile created for app
- Image built & pushed to Docker Hub
- Image pulled using Docker Compose
- MySQL data persisted with Docker volume
- Custom Docker network used

> App source code is forked.  
> Dockerization & deployment done by me.

## Run
```bash
docker compose up -d
